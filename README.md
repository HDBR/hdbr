### 🧑‍💻 Sobre mim

Sou o **Hildelbrando**, fundador da **HDBR Studios**. Trabalho na interseção entre **infraestrutura cloud**, **automação de processos** e **inteligência artificial**. Gerencio múltiplos servidores (Hetzner, Oracle Cloud, Hostinger) com Docker Swarm, construo fluxos de automação complexos com N8N, e desenvolvo soluções web e APIs com TypeScript e Python.

Minha filosofia é **self-hosting first**: controle total, custo otimizado e soberania dos dados.

- 🏢 **HDBR Studios** — DevOps, Automação & IA
- 🌍 Brasil
- 🐍 Python para IA & APIs | 🟦 TypeScript para web full stack
- 🐳 Docker Swarm em produção com Traefik + Portainer
- 🤖 Agentes de IA com Claude, GPT e OpenClaw
